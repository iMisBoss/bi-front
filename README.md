# OA系统 - 前端项目

## 📋 项目简介

企业OA管理系统前端项目，由**个人独立开发**完成的完整办公自动化界面解决方案。

> ⚠️ **重要声明**：本项目**仅为前端静态原型**，未对接任何后端接口，不具备商用、实操能力，仅作为前端页面展示使用。

## 🛠️ 技术栈

- **核心框架**：Vue 3.x
- **UI组件库**：Element Plus
- **路由管理**：Vue Router 4.x
- **状态管理**：Pinia
- **构建工具**：Vite
- **样式预处理**：SCSS

## 🚀 快速开始

### 环境要求

- Node.js >= 16.0.0
- npm >= 8.0.0

## 运行方式
1. 环境：Node.js 16+
2. 安装依赖：npm install
3. 本地运行：npm run dev
4. 打包：npm run build

## ⚠️ 项目局限性说明

### 🔴 当前状态

本项目是**个人独立开发的前端静态原型**，存在以下限制：

1. **无后端支持**
    - ❌ 未实现任何后端接口
    - ❌ 无数据库设计
    - ❌ 无服务器端逻辑

2. **无数据交互**
    - ❌ 所有数据均为静态Mock数据
    - ❌ 表单提交无实际效果
    - ❌ 查询操作返回固定数据

3. **无身份认证**
    - ❌ 登录功能仅为页面跳转
    - ❌ 无Token验证机制
    - ❌ 无权限控制逻辑

4. **非生产就绪**
    - ❌ 不具备商用条件
    - ❌ 不具备实际操作能力
    - ❌ 仅用于前端展示和演示

### 🟡 适用场景
